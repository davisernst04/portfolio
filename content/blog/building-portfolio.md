# Building My Portfolio

In this post, I'll share the tech stack and architecture of this portfolio.

## Tech Stack
- **Next.js**: The React framework for production.
- **Tailwind CSS**: For rapid and clean styling.
- **TypeScript**: Ensuring type safety across the project.
- **Lucide Icons**: Beautiful and consistent icons.

## Features
1. **Bento Grid**: Showcasing projects and skills in a modern layout.
2. **Draggable Cards**: Adding interactivity to the UI.
3. **Blog System**: Sharing my thoughts and experiences.

I'm really excited about how this is turning out!
